# VS Code Useful Cheat Sheet (Pro)

## Navigation

- `Ctrl+P` → Quick Open file
- `Ctrl+T` → Go to symbol in workspace
- `Ctrl+Shift+O` → Go to symbol in file
- `Ctrl+G` → Go to line

## Refactor & Code Intelligence

- `F12` → Go to definition
- `Alt+F12` → Peek definition
- `Shift+F12` → Find references
- `F2` → Rename symbol
- `Ctrl+.` → Quick fix / code actions

## Editing Speed

- `Ctrl+D` → Select next same word
- `Ctrl+Shift+L` → Select all occurrences
- `Alt+Up/Down` → Move line
- `Shift+Alt+Up/Down` → Duplicate line
- `Ctrl+/` → Toggle line comment

## Search

- `Ctrl+F` → Find in file
- `Ctrl+H` → Replace in file
- `Ctrl+Shift+F` → Search in workspace
- `Ctrl+Shift+H` → Replace in workspace

## Layout / Panels

- `Ctrl+B` → Toggle sidebar
- `Ctrl+J` → Toggle panel
- `` Ctrl+` `` → Toggle terminal
- `Ctrl+\` → Split editor
- `Ctrl+1/2/3` → Focus editor group

## Debugging

- `F5` → Start/Continue
- `F9` → Toggle breakpoint
- `F10` → Step over
- `F11` → Step into
- `Shift+F11` → Step out

## Git

- `Ctrl+Shift+G` → Source control
- `Ctrl+Enter` (in commit box) → Commit
- `...` in SCM panel → Push/Pull/Sync actions

## Custom control layer (your setup)

- `Ctrl+Alt+E` → Explorer
- `Ctrl+Alt+T` → Terminal
- `Ctrl+Alt+M` → Problems
- `Ctrl+Alt+R` → Rename
- `Ctrl+Alt+Z` → Zen mode
- `Ctrl+Alt+B/L/Y` → Build/Lint/Typecheck tasks
